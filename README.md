# Creatify

A feature-rich Android drawing application built using Kotlin and Android Canvas APIs.

## Features
- Freehand Drawing
- Brush Size & Opacity Control
- Eraser Tool
- Shape Tools
  - Line
  - Rectangle
  - Circle
  - Triangle
  - Arrow
- Bucket Fill Tool
- Undo / Redo
- Background Colors
- Background Images
- Save Drawing
- Share Drawing

## Tech Stack
- Kotlin
- Android SDK
- Canvas & Paint API
- Coroutines
- PopupWindow
- Material Components
